# 🤖 DeepSeek 提示詞審查與優化

## 觸發條件
用戶說「審查這個 prompt」、「優化提示詞」、「DeepSeek 優化」或貼 prompt 草稿時觸發。

## 功用
專為 DeepSeek V4 Flash/Pro 特化的提示詞審查與重構工具，最大化 DeepSeek 的深度推理與長上下文能力。同時確保跨模型（GPT、Gemini、Claude）相容性。

## 使用方式
貼上你的 prompt 草稿，說「幫我審查這個」。

## 輸出格式
- 🔍 問題點（2-4 個致命缺陷）
- ✅ 優化版提示詞
- 📊 改進說明對比表
- ⚠️ 模型相容性備註
