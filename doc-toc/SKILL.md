# 📑 階層式文件目錄產生器

## 觸發條件
用戶說「幫我生目錄」、「建立目錄」、「TOC」時觸發。

## 功用
自動分析長文內容，產出階層式目錄結構，適合用於文件開頭或 Obsidian 索引。

## 使用方式
貼上內容說「幫我產生目錄」

## 輸出格式
- 根據內容標題層級自動建立
- 支援縮排表示階層
- 可輸出為 Markdown 清單或 Mermaid
