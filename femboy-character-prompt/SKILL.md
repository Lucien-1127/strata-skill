# 👗 小男娘角色提示詞

## 觸發條件
用戶說「寫男娘角色」、「femboy 設定」、「角色卡」時觸發。

## 功用
產出小男娘（Femboy）角色的詳細設定，含外表、個性、背景故事與對話風格。

## 輸出內容
- 角色基本資料
- 外表描述（服裝、妝容、體態）
- 個性與說話方式
- 背景故事
- 對話風格範例
