# 合約範本庫

台灣民間實務合約範本，由用戶搜集、清理、留存。每份範本含：
- 原始來源（PDF／照片／連結）
- 清理後的正確文字版
- 可執行的 Python DOCX 產生器

## 使用方式

1. 從參考文件索引挑選合約類型
2. 讀取對應 markdown 確認條款
3. 執行對應 Python 腳本產生 DOCX
4. LibreOffice 轉 PDF
5. 排版審計後交付

## 鐵則

- **不猜字、不填空**：來源不明的條文不要自行推測，標記為待確認
- **以校正版為準**：使用者後續提供的校正版文字取代所有前期版本
- **逐條比對**：每條條文生成後與來源文字逐字比對
- **產出後審計**：QC 閘門包含錯字掃描、不當用語檢查、格式驗證
