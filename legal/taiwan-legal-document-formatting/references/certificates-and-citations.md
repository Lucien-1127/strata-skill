# 電子憑證網址備忘錄

## MOICA（內政部憑證管理中心）

| 項目 | 內容 |
|:-----|:------|
| 官網 | `moica.nat.gov.tw` |
| 角色 | 自然人憑證（MOEACA）簽發機構 |
| 架構 | GPKI（政府憑證管理中心）第一層下屬憑證機構 |
| 用途 | 網路地政電子謄本、線上申辦土地登記、稅務申報 |
| 注意 | `moea.nat.gov.tw` 是經濟部，**非**憑證中心 |

## 行動自然人憑證

| 項目 | 內容 |
|:-----|:------|
| 官網 | `fido.moi.gov.tw` |
| 角色 | 行動版自然人憑證（手機載具） |
| 用途 | 線上簽署 NDA、電子簽章合約 |
| 與 MOICA 區別 | 不同入口、不同驗證流程，不可混淆 |

## 簽證基金相關

地政士法第22條第3項授權訂定之「地政士簽證責任及簽證基金管理辦法」，簽證基金賠付上限 `新臺幣400萬元/人` 係規定於該辦法，非地政士法§22本文。
