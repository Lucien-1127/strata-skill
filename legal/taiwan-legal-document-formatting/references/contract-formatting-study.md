# 台灣民間借款抵押權設定契約書 — 格式研究

> 研究日期：2026-07-08
> 來源：全國法規資料庫、地政司實務慣例、合約軌道規範

## 格式規範速查

| 項目 | 規格 |
|:-----|:------|
| 紙張 | A4，上3cm/左右下2.5cm |
| 字型 | 標楷體（cwTeXKai）12pt |
| 英數字 | Times New Roman 12pt |
| 標題 | 18-20pt 粗體，置中 |
| 條號 | 12pt 粗體，前後空行 |
| 內文 | 12pt，縮排2字 |
| 行距 | 固定22pt |
| 頁碼 | 頁尾中央，10pt |

## 條款層級縮排

```
第X條（粗體，靠左，前後空行）
  項次不冠數字，直接接續，縮排2字
    一、二、三（中文數字，縮排2字）
      (一)(二)（縮排4字）
```

## 第一頁常見錯誤

- 甲方/乙方資訊與副標題同行 → 應獨立「立契約書人」段落分行
- 簽名欄用底線填空 → 應用地政標準表格（Table Grid）
- 注意事項用 emoji → 法律文件禁用，用標準文字「注意事項」
- 無頁碼 → 頁尾中央
- 金額無中文大寫 → 中文大寫 + 括號阿拉伯數字

## 利息寫法

民法§205 利率上限年息16%。月息2% = 年息24%已逾上限，超過部分無請求權。
合約應同時載明約定利率 + 民法§205但書。
