## 排版規格（抵押權設定契約書專用）

**當契約用於地政機關登記時，本節規格取代主技能中商業合約軌道的通用設定。**

| 項目 | 規格 | 理由 |
|:-----|:------|:------|
| 紙張 | A4，邊界 3cm(上) / 2.5cm(左右下) / 2.8cm(裝訂側) | 裝訂側加寬供騎縫章 |
| 字型 | **標楷體**（Linux: cwTeXKai） | 地政/法院審查慣例，不可用 Noto/文泉驛 |
| 內文字級 | 12pt，固定行距 22pt | 標準閱讀舒適度 |
| 條號字級 | 12pt **粗體** | 與內文同字級但加粗區別 |
| 標題字級 | 18~20pt 粗體居中 | 文件主標題 |
| 英數字（日期/NT$） | Times New Roman 12pt | 搭配中文字型 |
| 頁碼 | 頁尾居中阿拉伯數字 | 多頁文件必備 |
| 簽名欄 | Table Grid 雙欄表格 | 文字底線**禁止使用** |

**字型陷阱**：雖然主技能商業合約軌道建議 Noto Sans CJK TC，但抵押權設定契約書需送地政機關登記，標楷體是地政/法院慣例。用 Noto 可能導致地政事務所審查時要求修改。

### 第一頁布局規則（🚨 最常出錯）

```
金錢消費借貸暨抵押權設定契約書          ← 18~20pt 粗體居中
（普通抵押權 · 第一順位）                ← 14pt 居中

立契約書人                               ← 12pt 獨立段落

甲方（貸與人）：_________
身分證統一編號：_________
住址：_________________________________
電話：_________

乙方（借用人）：_________
身分證統一編號：_________
住址：_________________________________
電話：_________

雙方為金錢消費借貸及抵押權設定事宜，
同意訂立本契約書，條款如下：             ← 契旨段落

第一條　借款金額                         ← 條號開始
```

**常見錯誤：**
- 甲方/乙方資訊放在標題副標同行 → 應獨立成段落
- 缺少「立契約書人」過渡 → 突兀
- 第一條與當事人資訊之間無間距 → 排版擠壓

### 條號層級格式（標楷體 12pt）

```
第Ｘ條　條文標題         ← 靠左粗體，前空12pt後空2pt
                          條號與標題間用全形空格（　）

  內文（項次不冠數字）   ← 縮排 2 字（約 0.7cm）

  一、款次（中文數字）    ← 縮排 2 字
    (一) 目次（括號數字） ← 縮排 4 字
      1. 細目（阿拉伯）   ← 縮排 6 字
```

**條號與標題間距：**
- ✅ `第一條　借款金額`（全形空格）
- ❌ `第一條 借款金額`（半形空格）
- ❌ `第一條借款金額`（無空格）

### 簽名欄表格規格

必須使用 **Table Grid 雙欄表格**，不可使用文字底線。格式如下：

```
┌──────────────────────────────┬──────────────────────────────┐
│ 甲方（貸與人）：             │ 乙方（借用人）：             │
│ 姓名：_______________        │ 姓名：_______________        │
│ 身分證統一編號：___________  │ 身分證統一編號：___________  │
│ 住址：_______________        │ 住址：_______________        │
│ 電話：_______________        │ 電話：_______________        │
│ 簽名或蓋章：_______________  │ 簽名或蓋章：_______________  │
├──────────────────────────────┴──────────────────────────────┤
│              中　　華　　民　　國　○年　○月　○日            │
└──────────────────────────────────────────────────────────────┘
```

- 欄寬各 8cm（A4 扣邊距後約 16cm）
- 最後一列合併為單列，置中，日期字距加大
- 表格上方空一行，加「簽　署　頁」居中標題

### 金額寫法

```
新臺幣伍拾萬元整（NT$500,000）
```

中文大寫在前，阿拉伯數字含貨幣代碼在後括號內，必須有「整」字。

## 費用標準

| 項目 | 金額 |
|:-----|:------|
| 設定登記費 | 擔保金額 × 千分之一 |
| 他項權利證明書 | 80元/張 |
| 代書費（地政士報酬） | 4,000 ~ 6,000元 |
| 印花稅 | 擔保金額 × 千分之一 |
| 地政規費（謄本等） | 依實際發生 |

## 注意事項

- 民法§205（2023年修正）：約定利率超過週年**16%**者，超過部分無效。契約範本應預設年息16%。
- 簽名欄用表格而非文字底線（地政機關不接受文字底線格式）
- 第一頁甲方/乙方資訊必須獨立為段落，不可與標題混排
- 每條條文前加 `^[民法§XXX]` 格式引用（G3標準）— 僅用於關鍵條文
- 生成腳本參考：`scripts/generate_doc.py`

## Pitfalls

- ❌ 2026-07-08：合約第一頁甲方/乙方資訊與標題混排 — 用戶最常退回的格式問題。必須將當事人資訊獨立為「立契約書人」後分列段落。
- ❌ 2026-07-08：簽名欄使用文字底線填空 — 地政機關審查不接受，必須使用 Table Grid 表格。
- ❌ 2026-07-08：使用 Noto Sans CJK TC 而非標楷體 — 抵押權設定契約書需送地政機關登記，應優先使用標楷體。
- ❌ 2026-07-08：注意事項使用 emoji（⚠️）開頭 — 法律文件禁止使用 emoji，應改為「注意事項」標準文字。
- ❌ 2026-07-08：契約利率預設月息2%（年息24%）— 已逾民法§205之16%上限，應預設為年息16%。
