# 預告登記關鍵判決（5 則）

> 產生日期：2026-07-07
> 來源：司法院法學資料檢索系統 judgment.judicial.gov.tw

## 1. 最高法院95年度台上字第2323號 — 預告登記塗銷（§767）

| 項目 | 內容 |
|:-----|:------|
| 裁判日期 | 2006.10.26 |
| 案由 | 塗銷預告登記事件 |
| 爭點 | 預告登記同意書效力；所有權人依§767請求塗銷之要件 |
| 見解 | 若預告登記所保全之請求權已不存在，登記名義人得依§767請求塗銷；若基於有效同意書辦理，不能逕以同意書得撤銷而主張塗銷 |

## 2. 最高法院103年度台上字第1038號 — 預告登記與時效（§125）

| 項目 | 內容 |
|:-----|:------|
| 裁判日期 | 2014.06.05 |
| 案由 | 塗銷預告登記等 |
| 爭點 | 消滅時效；時效完成後預告登記之效力 |
| 見解 | 時效依原請求權性質定之（一般15年）；預告登記不延長時效；時效完成後非經塗銷仍存在；時效抗辯成立時得請求塗銷 |

## 3. 最高法院92年度台上字第1309號 — 詐欺撤銷（§92/§93）

| 項目 | 內容 |
|:-----|:------|
| 裁判日期 | 2003.06.19 |
| 案由 | 確認本票債權不存在等 |
| 爭點 | 受詐欺簽署同意書之撤銷 |
| 見解 | 同意書簽署為法律行為；得依§92於發現詐欺後一年內撤銷；撤銷後同意書自始無效，預告登記失所附麗 |

## 4. 最高法院108年度台上字第2162號 — 借名登記

| 項目 | 內容 |
|:-----|:------|
| 裁判日期 | 2019.09.19 |
| 案由 | 請求移轉土地所有權等 |
| 爭點 | 借名登記中預告登記效力 |
| 見解 | 出名人未經借名人同意之預告登記可能違反誠信原則或構成權利濫用而無效；借名人得終止借名關係後請求塗銷 |

## 5. 臺灣高等法院110年度上易字第1167號 — 地政士刑事責任（§214）

| 項目 | 內容 |
|:-----|:------|
| 裁判日期 | 2021.08.17 |
| 案由 | 偽造文書 |
| 爭點 | 地政士明知文件不實仍申辦之刑事責任 |
| 見解 | 地政士負實質審查義務（地政士法§18/§22）；明知不實仍申辦構成§214（三年以下）；可能另涉§215/§216/§339 |
