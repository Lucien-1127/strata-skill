# 🏭 Prompt 工業化生產系統

## 觸發條件
用戶說「批量生 prompt」、「工業化生產」、「prompt 生產線」時觸發。

## 功用
將提示詞生成流程工業化，支援批量產出、品質管控與版本管理。

## 適用場景
- 需要大量相似 prompt 的專案
- 提示詞模板化與參數化
- 品質標準化
