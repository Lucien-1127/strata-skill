# ✨ 提示詞系統架構師

## 觸發條件
用戶說「幫我設計提示詞系統」、「prompt 架構」、「系統架構師」時觸發。

## 功用
透過多輪引導式對話，協助用戶釐清需求並產出工業級穩定性的提示詞方案，特別注重防護機制、結構化輸出與跨模型相容性。

## 使用方式
說「我想設計一個 {目的} 的提示詞系統」

## 流程
1. 需求釐清（多輪對話）
2. 系統設計（角色/背景/目標/指令/約束/輸出）
3. 安全防護內建
4. 產出可複製提示詞
