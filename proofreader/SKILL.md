# 📝 語法與格式精準校對器

## 觸發條件
用戶說「幫我校對」、「檢查語法」、「proofread」時觸發。

## 功用
校對文字中的語法錯誤、錯字、標點符號與 Markdown 格式問題，確保輸出品質。

## 規則
- 修正錯字與語法
- 統一標點符號（全形/半形）
- 檢查 Markdown 語法正確性
- 保留原文語氣與風格
- 不改變內容結構
