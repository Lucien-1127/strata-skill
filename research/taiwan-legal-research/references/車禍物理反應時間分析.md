# 車禍物理反應時間分析

> **匯集本事故分析中發展出的物理與反應時間分析模式**
> 適用於「追撞事故」及「突發障礙物事件」中判斷駕駛人有無物理上之反應可能性

## 核心物理公式

### 基礎計算

```
可用反應時間 t = 可用距離 d ÷ 相對速度 v

t (秒) = d (公尺) / v (m/s)
```

### 車速換算

| 車速 (km/h) | 車速 (m/s) | 換算公式 |
|:-----------:|:----------:|:---------|
| 10 km/h | 2.78 m/s | × 1000/3600 |
| 20 km/h | 5.56 m/s | × 0.2778 |
| 30 km/h | 8.33 m/s | × 0.2778 |
| 40 km/h | 11.11 m/s | × 0.2778 |
| 50 km/h | 13.89 m/s | × 0.2778 |
| 60 km/h | 16.67 m/s | × 0.2778 |
| 70 km/h | 19.44 m/s | × 0.2778 |

### 已知距離求可用時間

```
t(秒) = d(m) × 3.6 / v(km/h)
```

### 已知時間求最短安全距離

```
d(m) = v(km/h) × t(秒) / 3.6
```

## 人類感知反應時間（PRT）標準值

| 情境 | PRT (秒) | 說明 | 出處 |
|:-----|:--------:|:------|:-----|
| **一般駕駛人** | **0.75~1.0秒** | 認知+決策+動作之標準值 | 交通工程手冊、ITE / AASHTO |
| 職業駕駛人（已預警） | 0.5~0.7秒 | 競技/職業駕駛，且已有預期 | 運動視覺研究 |
| 年輕駕駛人 | 0.64~0.85秒 | 20~30歲族群 | NHTSA 研究 |
| 年長駕駛人 | 1.0~1.5秒 | 65歲以上族群 | NHTSA 研究 |
| 分心駕駛（打電話） | 1.2~1.8秒 | 認知負載增加 | 多項實證研究 |
| 夜間行駛 | 增加0.2~0.5秒 | 低光源環境 | AASHTO 綠皮書 |

### PRT 的三階段分解

```
┌─────────────┬──────────────┬──────────────┐
│   感知       │    認知      │    動作      │
│ (眼睛接收)   │ (大腦辨識)   │ (執行操作)   │
├─────────────┼──────────────┼──────────────┤
│ ~0.1~0.2秒  │ ~0.3~0.5秒   │ ~0.2~0.4秒   │
│ 眼球追蹤物體│ 辨識為何物   │ 鬆油門→移至 │
│ 視網膜成像  │ 決定是否煞車 │ 煞車踏板→踩 │
│             │ 選擇閃避路線 │ 或轉方向盤   │
└─────────────┴──────────────┴──────────────┘
```

### 在 0.36~0.45 秒內常人可能完成的動作

| 時間 | 可完成 | 不可完成 |
|:----:|:-------|:---------|
| 0.36~0.45秒 | 眼球追蹤物體(~0.1s) | ❌ 完整辨識倒地機車為何物 |
| | 大腦初步警覺「有東西」(~0.15s) | ❌ 判斷是否需要煞車或閃避 |
| | 開始移動腳至煞車踏板(~0.2s) | ❌ 完成煞車減速 |

## 追撞事故標準分析表

### 後車反應時間 vs 車速對照表

| 可用距離 | 30 km/h (8.33 m/s) | 40 km/h (11.11 m/s) | 50 km/h (13.89 m/s) | 60 km/h (16.67 m/s) |
|:---------|:------------------:|:-------------------:|:-------------------:|:-------------------:|
| **3 m** | 0.36 s ❌ | 0.27 s ❌ | 0.22 s ❌ | 0.18 s ❌ |
| **5 m** | **0.60 s** ❌ | **0.45 s** ❌ | **0.36 s** ❌ | **0.30 s** ❌ |
| **10 m** | 1.20 s ✅ | 0.90 s ⚠️ | 0.72 s ❌ | 0.60 s ❌ |
| **15 m** | 1.80 s ✅ | 1.35 s ✅ | 1.08 s ⚠️ | 0.90 s ⚠️ |
| **20 m** | 2.40 s ✅ | 1.80 s ✅ | 1.44 s ✅ | 1.20 s ✅ |
| **30 m** | 3.60 s ✅ | 2.70 s ✅ | 2.16 s ✅ | 1.80 s ✅ |

> ✅ = 可完成感知-反應-煞車完整流程 (t ≥ 1.0s)
> ⚠️ = 有困難但可能 (0.75s ≤ t < 1.0s)
> ❌ = 物理上不可能 (t < 0.75s)

### 緊急煞車所需總距離（含反應距離）

```
總煞停距離 = 反應距離(v×t_prt) + 實際煞車距離(v²/2μg)

μ = 輪胎與路面摩擦係數（乾燥瀝青約0.7~0.8，潮濕約0.4~0.6）
g = 9.81 m/s²
```

| 參數 | 40 km/h | 50 km/h |
|:-----|:-------:|:-------:|
| 反應距離 (1.0秒 PRT) | 11.1 m | 13.9 m |
| 煞車距離 (μ=0.75) | 8.4 m | 13.1 m |
| **總煞停距離** | **19.5 m** | **27.0 m** |

> 按此標準，在 40~50km/h 車速下，僅有約 5m 的可用距離 → 總煞停距離 (19.5~27m) 遠超過可用距離。

## 車輛倒地後法律定位與物理時間線分析

### 三階段模型

```
時序線：

t₋₁：機車紅燈右轉（行駛中）
t₀：  機車因高低差開始不穩（行駛中→失控）  ← 汽車可能尚未看見
t₁：  機車完全倒地（脫離行駛中車輛地位）   ← 汽車若此時才發現，已無反應時間
t₂：  機車因慣性滑入/躺在車道中（障礙物） ← 碰撞發生或即將發生
t₃：  碰撞發生
```

### 階段 A（t₋₁ → t₀）：行駛中車輛階段

- 機車仍為交通參與者
- 適用前後車關係（§94①）與轉彎車讓直行車（§102①⑦）
- 信賴原則：汽車可主張但因§94③仍有限制

### 階段 B（t₀ → t₁）：倒地轉換期

- 機車駕駛人喪失車輛控制能力
- 機車非「行駛中車輛」亦非完全「障礙物」 — 法律灰色地帶
- 此階段通常極短（<0.5秒）

### 階段 C（t₁ → t₂）：道路障礙物階段

- 機車已靜止或在地面滑行，駕駛人無法控制
- 法律定位：道路臨時障礙物（道安規則§93①②「道路發生臨時障礙」）
- ❌ §94①前後車安全距離推定過失**不適用**
- 回歸§94③一般注意義務（注意車前狀況並採取必要措施）
- 信賴原則適用空間**增大**（因他方已非交通參與者）

## 追撞已倒地物體 vs 追撞行駛中車輛：法律評價對照

| 比較項目 | 追撞行駛中車輛 | 追撞已倒地物體（障礙物） |
|:---------|:--------------|:------------------------|
| **法律定位** | 前後車關係 | 障礙物／道路臨時障礙 |
| **安全距離義務(§94①)** | ✅ 適用（推定過失） | ❌ 不適用 |
| **注意車前狀況(§94③)** | ✅ 適用 | ✅ 適用（一般義務） |
| **減速義務(§93①②)** | 路口有義務 | ✅ 遇道路臨時障礙應減速 |
| **信賴原則** | 可主張但受限 | **更易主張** |
| **可預見性** | 較高（前車可能煞車） | **較低**（車輛倒地非典型） |
| **因果關係** | 追撞行為 → 傷害 | 倒地原因 + 追撞之交互因果 |

## 二種倒地情境比較

### 情境一：先倒地 → 後滑入車道

```
時間軸示意：
  t₀: 機車倒地（在車道邊緣或機車道）
  ↓  可觀察到倒地到滑入車道之間有 ~0.5~1.0秒間隔
  t₁: 機車滑入車道
  t₂: 碰撞發生
```

- ✅ 汽車駕駛人更難以預見（倒地發生在車道外）
- ✅ 滑入車道的過程可能仍需 0.5~1.0 秒
- ✅ 信賴原則適用空間最大
- ✅ 若倒地發生在車道外（如機車道、路肩），汽車駕駛人對此階段無注意義務

### 情境二：倒地同時滑入車道

```
時間軸示意：
  t₀: 機車在轉彎中失控，倒地同時侵入車道
  t₁: 碰撞發生（幾乎與倒地同時）
```

- ⚠️ 倒地與侵入車道幾乎同時，汽車駕駛人從可能看到機車到碰撞時間更短
- ⚠️ 實務上鑑定委員會可能將此視為單一連鎖事件，而非分階段評價
- ⚠️ 汽車駕駛人可能主張「看到時已在車道上且無法反應」

## 反應時間不足之法律效果

### 刑法第14條（過失定義）

> 行為人雖非故意，但按其情節**應注意，並能注意，而不注意**者，為過失。
> 行為人對於構成犯罪之事實，雖預見其能發生而確信其不發生者，以過失論。

**關鍵要件**：「能注意」— 指物理上與心理上具有注意之可能性。

### 反應時間不足的法律論證鏈

```
1. 基礎事實：兩車距離 ~5m，汽車時速 40~50 km/h
2. 物理計算：t = 5 / (v × 1000/3600) = 0.36~0.45秒
3. 人類極限：感知反應時間下限 = 0.75秒
4. 對比：可用時間(0.36~0.45s) < 人類極限(0.75s)
5. 結論：按刑法§14，行為人欠缺「能注意」之要件
          → 不構成過失（無注意可能性）
```

### 監理站「5m可反應」說法之評價

監理站人員稱「5m是可做反應空間」— 此為法律評價（規範性判斷），非物理事實認定。於不同車速下：

- 若前車時速 **20km/h**：5m ÷ 5.56 = 0.9 秒 → 勉強可能反應（近人類極限）
- 若前車時速 **10km/h**：5m ÷ 2.78 = 1.8 秒 → 確定可反應
- 本事故（汽車時速 **40~50km/h**）：5m 提供 **0.36~0.45 秒** → 物理上不可能

**結論**：監理站說法若泛指一般狀況（不限車速）或許可成立，但代入本事故實際車速（40~50km/h）時，物理上**不成立**。
