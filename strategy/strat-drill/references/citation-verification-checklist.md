# G3 引用格式驗證檢查表（模型委員會審計標準）

產出法律 PDF 後，於模型委員會 (strat-drill) 審計引用時逐項檢查。

---

## 🔴 一、確定錯誤（7 項，任一項 FAIL 即不可交付）

| 編號 | 檢查項 | 通過條件 |
|:----:|:-------|:---------|
| E1 | 條文內容與現行法一致 | 引文須經 law.moj.gov.tw 比對，非憑記憶撰寫 |
| E2 | 條號與刑度配對正確 | §20 無刑罰、§171 才是刑責條，不可混用 |
| E3 | 來源類型正確 | 學說歸學說、法條歸法條、判決歸判決 |
| E4 | 條文引用精確 | §92(得撤銷) vs §93(除斥期間一年) — 不可混淆 |
| E5 | 數字來源可追溯 | 具體數字（如400萬）須出自條文或管理辦法，非推測 |
| E6 | 網址正確 | moea(經濟部) vs moica(內政部憑證) 差一個字母即錯誤 |
| E7 | 引文版本為現行條文 | 不得引用舊版條文（如證交法§20缺「私募或買賣」） |

**判定規則：** E1~E7 任一項未通過 → 🔴 紅燈，退回修正後再審。

## 🟡 二、引用品質（4 項）

| 編號 | 檢查項 | 通過條件 |
|:----:|:-------|:---------|
| Q1 | 判決字號可追蹤 | 實務見解引用應附字號；無字號時降級為「有力見解」 |
| Q2 | 來源層級明確 | 法律 / 行政規則 / 判決 / 學說 應分層標記 |
| Q3 | 學說非法律 | 學說定性（如「債權物權化」）不可掛 ^[全國法規資料庫] |
| Q4 | 引用前後一致 | 同一來源在不同章節的標記方式應一致 |

**判定規則：** Q1~Q4 未通過項 ≥2 → 🟡 黃燈，修正後重審。1項可記錄後交付。

## 🟢 三、G3 格式（4 項）

| 編號 | 檢查項 | 通過條件 |
|:----:|:-------|:---------|
| G1 | ^[來源簡稱] 格式 | 所有引用使用 ^[來源]，無舊式 [1][2] 殘留 |
| G2 | 【本段資料來源】區塊 | 每章/每節結束前有完整來源列表 |
| G3 | 彙總表完整 | 全文末尾有 G3 引用彙總表，涵蓋所有 ^[來源] |
| G4 | 來源是可查證的公開來源 | 不可引用私密文件、口頭陳述等不可查證來源 |

**判定規則：** G1~G4 全數通過 → 🟢 可交付。任一項 FAIL → 🟡 修正後再審。

## 常見陷阱（來自實戰審計）

| 陷阱 | 案例 | 教訓 |
|:-----|:------|:------|
| 法條刑度張冠李戴 | 證交法§20無刑罰，違反者歸§171 3~10年 | 條號有無刑罰要分清楚 |
| 除斥期間條號錯置 | 民法§92寫成包含一年時效 → 一年時效是§93 | 每個條號的精確範圍查 law.moj.gov.tw |
| 管理辦法與法律混淆 | 地政士§22沒有400萬數字 → 數字在簽證基金管理辦法 | 引用需精確到法規層級 |
| 網址差一個字母 | moea(經濟部) vs moica(內政部) | 政府網址確認後再輸出 |
| 學說結論掛法條來源 | 「債權物權化是學說」掛 ^[全國法規資料庫] | 學說歸學說，法條歸法條 |
| 實務見解無字號 | 「實務多數見解認為…」無判決字號 | 無字號時降級措辭「有力見解」 |
