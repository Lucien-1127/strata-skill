# Drill 別名：模型委員會 (Model Committee)

用戶慣用稱呼 **「模型委員會」** 對應 `strat-drill` 技能。

## 觸發方式
- 用戶說「送模型委員會 裁決」→ 載入 `strat-drill` 執行完整壓力檢核
- 用戶說「送模型委員會 裁決引用」→ 載入 `strat-drill`，重點檢核引用格式（G3 `^[來源簡稱]`）

## 標準流程
1. Anchor+5D 完成產出
2. 載入 `strat-drill`，執行：反例測試 → 邊界探索 → 缺口標記 → 燈號決策
3. 產出 Drill 報告（含 🟢/🟡/🔴 燈號）
4. 若 🟢 綠燈 → 進入 Forge 階段
5. 若 🟡/🔴 → 依燈號決定退回 Anchor 或補充分析

## 引用審查要點（模型委員會專用）
- 檢查文中 `^[來源簡稱]` 格式正確
- 檢查每個 ^[來源] 在【本段資料來源】有對應條目
- 檢查無舊式 [1][2] 殘留
- 檢查語氣中性不預設違法（用「可能涉及」取代「構成」）
