# 🎬 影片提示詞架構師工作流

## 觸發條件
用戶說「寫影片 prompt」、「影片腳本」、「AI 影片」時觸發。

## 功用
從概念到產出完整的影片生成提示詞工作流，支援 Seedance / 即夢 / Gemini 等平台。

## 執行流程
1. 需求訪談（主題/風格/長度）
2. 分鏡腳本設計
3. 逐鏡頭提示詞生成
4. 參數優化建議
