# 🌐 Web Clipper 知識內化模板

## 觸發條件
用戶貼了一個網址說「幫我剪輯」、「內化這個頁面」時觸發。

## 功用
將網頁內容剪輯為結構化知識筆記，並自動產出 Mermaid 心智圖，便於 Obsidian 管理。

## 執行步驟
1. 用 web_extract 抓取網頁內容
2. 萃取核心重點
3. 產出結構化筆記（含 Metadata）
4. 產出 Mermaid mindmap
