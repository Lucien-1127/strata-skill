# 🇹🇼 繁體中文在地化翻譯器

## 觸發條件
用戶說「幫我翻譯」、「轉繁體中文」、「在地化」或貼外文內容時觸發。

## 功用
將各類文本（中英日韓等）精準翻譯為台灣繁體中文（zh-TW），保留所有 Markdown 語法、雙向連結與原始語氣。

## 使用方式
直接貼內容給我，或說「翻譯這篇：{內容}」

## 執行步驟
1. 語系轉換：翻譯成繁中，優先使用台灣用語（網路/軟體/滑鼠）
2. 語境對齊：保留原文語氣（學術/商務/輕鬆）
3. 格式保護：保留 `[[` `]]` `#` `**` 等 Markdown 語法
4. 詞彙選擇：科技術語用台灣業界通用譯名
